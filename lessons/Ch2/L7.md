# Logarithms

<blockquote style="background-color: #041b2d; border-left: 5px solid #44a1ea; padding: 5px 10px; margin: 10px auto">
A logarithm is the inverse of an exponent.
</blockquote>

<blockquote style="border-left: 5px solid #6c7db0; padding: 5px 10px; margin: 10px auto">
24 = 16

log216 = 4

"log216" can be read as "log base 2 of 16", and means "the number of times 2 must be multiplied by itself to equal 16".

"log216" might also be written as log2(16)
</blockquote>

Some more examples:

|   Logarithm   |	Result  |
|   --- |   :---: |
|   log<sub>2</sub>2	|   1   |
|   log<sub>2</sub>4	|   2   |
|   log<sub>2</sub>8	|   3   |
|   log<sub>2</sub>16	|   4   |
|   log<sub>10</sub>10	|   1   |
|   log<sub>10</sub>100	|   2   |
|   log<sub>10</sub>1000	|   3   |
|   log<sub>10</sub>10000	|   4   |

### Python Syntax

There isn't a language-level operator to calculate a logarithm, but we can `import` the [`math`](https://docs.python.org/3/library/math.html) library and use the `math.log()` function.

```python
import math

print(f"Logarithm base 2 of 16 is: {math.log(16, 2)}")
# Logarithm base 2 of 16 is: 4.0
```
