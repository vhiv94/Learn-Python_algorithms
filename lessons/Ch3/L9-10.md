# Constants Quiz

Consider the following functions:

```python
def sum(nums):
    total = 0
    for num in nums:
        total+=num
    return total
```

```python
def double_sum(nums):
    total = 0
    for num in nums:
        double = num + num
        total += double
    return total
```
