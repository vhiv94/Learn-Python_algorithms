# Longest Common Prefix

Let's use our Trie to find the longest common prefix of a list of words. This feature can be used in LockedIn to display suggestions when users are searching for their connections.

Assignment
Complete the `longest_common_prefix` method. It returns the longest common prefix among the words in the trie.

1. [x] Initialize a variable `current` that references the root of the trie
2. [x] Initialize a variable `prefix` to an empty string
3. [x] Enter a forever `while` loop:
   1. [x] Get the "children" (keys) in the `current` dictionary
   2. [x] If a child is an end_symbol, break the loop.
   3. [x] If there is only one child, append the character to the prefix string and update the current dictionary to point to the child dictionary corresponding to the character.
   4. [x] If there are multiple children or no children, break the loop.
4. [x] Return the prefix string.
